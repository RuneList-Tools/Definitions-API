## About The Project

A definition Loader made for RSPS that plugins into your Server

Features
* Update when new Revs
* Warns you if you are going to overwrite existing files
* Allows you to ignore certain ids
* OSRS and Pre EOC

### Built With
* [Kotlin](https://kotlinlang.org/)
